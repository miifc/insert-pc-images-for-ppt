## insert-pc-images-for-ppt

PPT macro to capture and paste all the images in selected folder.
When pasting
* One image on one slide.
* Unify the width of the image from the slide width.
* Cropped so as not to protrude from the slide.

選択フォルダ内の画像を全て取り込んで貼り付けるPPTマクロ。
貼り付ける際に
* 1スライドに1画像。
* スライド幅から画像の幅を統一する。
* スライドからはみ出してしまわないようにトリミングする。
